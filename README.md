## 主題方向參考

### 🔸 1. 科學園區用電效率分析 ✅（主力分析）

> 用電量 / 營業額 = 每億元產值的耗電量

- **竹科 vs 中科 vs 南科**：哪個園區效率最佳？
- **產業別比較**：半導體 vs 光電 vs 精密機械
- **趨勢**：效率是否逐年改善？

### 🔸 2. 科學園區 vs 全國電子業 ✅

- **佔比**：科學園區用電占全國電子零組件用電比例
- **行業別售電**：半導體相關行業在全國用電的地位

### 🔸 3. 用電結構分析 ✅

- **尖峰用電比例**：台電供需資訊 + 系統負載資料
- **夜間用電比例**：系統負載資料（10分鐘）可算出各時段分布
- **平均負載 vs 最大負載（負載率）**

可回答：哪些時段對電網壓力最大？是否出現「鴨子曲線」現象？

### 🔸 4. 穩定性分析 ✅

- **用電波動（標準差/變異係數）**：系統負載資料
- **備轉容量率歷年趨勢**：低備轉警戒天數統計（<10%）

### 🔸 5. 空間分析（QGIS）⚠️

> 全球電廠資料庫有台灣電廠座標，可作基礎空間視覺化

- 台灣電廠分布（燃料類型/裝置容量）← 全球電廠資料庫已有
- 工業區 vs 電廠距離 ← 缺工業區 Shapefile
- 區域供電壓力熱圖 ← 缺分區電網負載資料

**仍缺：**
- 工業區地理座標 / Shapefile → 工業局工業區資料（moeaidb.gov.tw）

### 🔸 6. 國際比較（OWID）✅

- 台灣工業用電密度 vs 鄰近國家（韓/日/新加坡）
- 再生能源滲透率國際定位
- 用電 / GDP 效率比較

## 四、建議路線

> **「科學園區用電效率 + 電網壓力 + 氣候影響」三合一**

**具體做法：**

1. **宏觀層**：台灣整體用電成長趨勢 + 發電結構轉型（再生能源滲透 1950–2025）
2. **供需平衡層**：備轉容量率趨勢 + 低備轉警戒日統計
3. **科學園區層**：
   - 竹科/中科/南科高壓用電趨勢（`vw_TechClimate_Base.csv`，2016–2025 全量）
   - 氣溫與高壓用電相關性（建議用 `vw_TechClimate_Analytics.csv` MoM 差分；安定需獨立處理）
   - 各產業用電效率（用電/營業額）
4. **決策層**：電網風險日曆 + 高耗能產業識別 + 再生能源導入時機
5. **能源效率層**：全國電力密集度 / 能源生產力長期趨勢（`esist_energy_index.csv`）
