## 主題方向參考

### ⚡ 主題 1（最推薦）—— 科學園區用電模式與電網壓力分析

**重點：** 科學園區高壓用電、備轉容量風險、氣候影響

| 項目 | 可用性 | 資料來源 |
|------|--------|---------|
| 竹科/中科/南科周邊高壓用電趨勢 | ✅ | `TechHubPower` (Azure SQL) |
| 氣溫 vs 高壓用電相關性 | ✅ | `ClimateData` + `TechHubPower` |
| 全國尖峰負載趨勢 | ✅ | 台電供需資訊 |
| 備轉容量率歷年變化 | ✅ | 備轉容量三年日資料 |
| 低備轉警戒天數統計（<10%）| ✅ | 可直接從備轉容量率計算 |
| 系統負載曲線（日內形狀/鴨子曲線）| ✅ | 10分鐘負載資料 |

### 🧩 主題 2（穩）—— 科學園區產業用電效率分析

**重點：** 用電量/營業額效率、產業差異

| 項目 | 可用性 | 資料來源 |
|------|--------|---------|
| 科學園區各產業用電效率（用電/營業額）| ✅ | 科學園區營業額 + 電子零組件用電 |
| 科學園區廠商數 vs 用電趨勢 | ✅ | 科學園區廠商數 + `TechHubPower` |
| 全國行業別用電排名 | ✅ | 行業別售電 2020–2024 |
| 電子/半導體占全國工業用電比 | ✅ | 行業別售電 + 電力消費月資料 |

### 🌱 主題 3（偏策略）—— 再生能源導入與電網風險

**重點：** 再生能源滲透、鴨子曲線、太陽能適配、能源安全

| 項目 | 可用性 | 資料來源 |
|------|--------|---------|
| 再生能源結構轉型趨勢（75年）| ✅ | 台電歷年發購電量（1950–2025）|
| 日間穩定用電 → 太陽能適配評估 | ✅ | 10分鐘負載資料 |
| 太陽光電各站發電量 | ⚠️ | 太陽光電 CSV（待匯入）|
| 國際再生能源比較 | ✅ | OWID 能源資料 |
| 進口能源依存度趨勢（2016–2025）| ✅ | 能源指標季資料（`esist_energy_index.csv`）|
| 進口原油來源集中度（HHI）| ✅ | 能源指標季資料（`esist_energy_index.csv`）|
| 發電能源種類集中度趨勢 | ✅ | 能源指標季資料（`esist_energy_index.csv`）|

### 🧠 主題 4（高階）—— 用電 × 產業 × 經濟貢獻效率四象限

**重點：** 高用電高產值 vs 低效區的四象限分析

| 項目 | 可用性 | 資料來源 |
|------|--------|---------|
| 科學園區三園四象限分析 | ✅ | 科學園區營業額 + 電子零組件用電 |
| 科學園區各產業四象限 | ✅ | 科學園區廠商數 × 產業別 + 電子零組件用電 |
| 全產業四象限分析（用電彈性版）| ⚠️ | 製造業生產指數（月×31產業）+ 電力消費月資料（8 產業可對接）| 生產指數非產值，四象限為「用電成長率 × 生產成長率」；對接欄位需手動對齊 |
| 全國電力密集度 vs 彈性值趨勢 | ✅ | 能源指標季資料（`esist_energy_index.csv`）| 全國總量層級，不拆產業；作為上述產業四象限的宏觀背景 |

## 四、建議路線

> **「科學園區用電效率 + 電網壓力 + 氣候影響」三合一**

**具體做法：**

1. **宏觀層**：台灣整體用電成長趨勢 + 發電結構轉型（再生能源滲透 1950–2025）
2. **供需平衡層**：備轉容量率趨勢 + 低備轉警戒日統計
3. **科學園區層**：
   - 竹科/中科/南科高壓用電趨勢（`TechHubPower`）
   - 氣溫與高壓用電相關性（`ClimateData` × `TechHubPower`）
   - 各產業用電效率（用電/營業額）
4. **決策層**：電網風險日曆 + 高耗能產業識別 + 再生能源導入時機
5. **能源效率層**：全國電力密集度 / 能源生產力長期趨勢（`esist_energy_index.csv`）
