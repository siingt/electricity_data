# 主題方向參考
> **「科學園區用電效率 + 電網壓力 + 氣候影響」三合一**

```
1. **宏觀層**：台灣整體用電成長趨勢 + 發電結構轉型（再生能源滲透 1950–2025）
2. **供需平衡層**：備轉容量率趨勢 + 低備轉警戒日統計
3. **科學園區層**：
   - 竹科/中科/南科高壓用電趨勢（`vw_TechClimate_Base.csv`，2016–2025 全量）
   - 子園區月峰值負載 + 負載率（`science_park_load.csv`，2016–2026）
   - 氣溫與高壓用電相關性（建議用 `vw_TechClimate_Analytics.csv` MoM 差分；安定需獨立處理）
   - 各產業用電效率（用電/營業額）
4. **產業效率層**：
   - 全國製造業 8 行業電力密集度（`electricity_intensity_annual.csv`，千度/百萬元，2007–2024）
   - GDP 貢獻度 vs 用電份額（識別高效 vs 高耗能行業）
5. **決策層**：電網風險日曆 + 節能優先行業識別 + 再生能源導入時機
6. **能源效率層**：全國電力密集度 / 能源生產力長期趨勢（`esist_energy_index.csv`）+ 人均耗能強度（主題5）
```
